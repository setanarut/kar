## Controls

| Key                   | Action            |
| --------------------- | ----------------- |
| WASD                  | Move              |
| Right shift           | Attack / Use      |
| Arrow keys            | Place block       |
| G                     | Toggle fly-Mode   |
| Hold C + Cursor click | Place rigid box   |
| O                     | Camera zoom out   |
| P                     | Camera zoom in    |
| Backspace             | Reset camera Zoom |
| V                     | Toggle debug mode |
